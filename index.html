给诸位大佬们准备了一节变速编辑教学说明页

从以下两步来告诉大家如何制作变速文件

准备工具 -- 编写变速





安卓/苹果用户使用malody编辑mc文件请准备一只笔一张纸或者一个最强大脑

Windows PC端用户使用malody编辑mc文件只需要一个windows记事本 或 visual studio code 和 malody客户端游戏即可

MacBook / iMac 用户需要准备一个可以运行malody的手机 和 atom / xcode





先完成你的作品，保证后续修改动作不会过多

然后开始对着你已经写好的谱面开始试听

把你认为需要变速的地方，记录在mc的

"effect":[]

这里




格式："effect":["beat":{0,2,4},"scroll":1]

什么意思呢

就是第一段的第四分之二处，1倍速播放














"effect":["beat":{4,3,4},"scroll":1.5]

这样 就是在第五段的第四分之三处，1.5倍速播放














"effect":["beat":{100,1,4},"scroll":-1]

这样 就是在第101段的第四分之一处，负1倍速播放
